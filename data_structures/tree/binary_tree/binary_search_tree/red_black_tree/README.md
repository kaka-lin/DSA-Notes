# 紅黑樹 (Red-Black Tree)

`紅黑樹 (Red-Black Tree)` 是一種`自平衡二元搜尋樹 (self-balancing binary search tree)`:

  - 比 [2-3-4 樹](../2_3_4_tree/)好 implement。
  - 平衡性要求比 [AVL Tree](../avl_tree/) 還寬鬆。
